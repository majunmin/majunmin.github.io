总结

总结下Starter的工作原理:

  1. Spring Boot在启动时扫描项目所依赖的JAR包，寻找包含 spring.factories 文件的JAR包
  2. 根据 spring.factories 配置加载AutoConfigure类
  3. 根据 @Conditional 注解的条件，进行自动配置并将Bean注入Spring Context
